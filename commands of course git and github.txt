commands of course git and github

pwd: show your folder
git config: check your Git configuration
git config --global user.name "Ruben": setup your Git username
git config --global user.email "r.cortez@accenture.com":  setup your Git user email
git init: To initialize a Git repo
git add : to add a file to the staging area in Git.
git add . : to add all files in the staging area in Git
git status: to check a repository's status in Git
git commit: to commit changes in the editor in Git.
git commit -m "": to commit changes with a message in Git
git log:  to see your commit history in Git
git log --stat:  to see log statistics in Git
git rm : to remove tracked files from the current working tree in Git
git diff: to see changes made before committing them using "diff" in Git
git checkout : to revert unstaged changes in Git.
git branch: to create a new branch in Git
git merge : to merge two branches in Git
git push: When all your work is ready to be saved on a remote repository, you can push all changes 
git pull:  If other team members are working on your repository, you can retrieve the latest changes made to the remote repository with the command
git fetch:  This command will download the changes from a remote repo but will not perform a merge on your local branch
git clone + URL: to clone an Existing Repository.
